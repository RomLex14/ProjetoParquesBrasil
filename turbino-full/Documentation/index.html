Setup Contact Form#back to top

  • Warning: You must have properly installed on your server the Theme, your server mail properly configurated, eg: with postfix ( But don't worry too much, most server do this automatically)
    Disclaimer: Your contact form will only work with your own mail server, meaning if your mail is hello@domain.com, your site must be domain.com
  • Step 1 - Locate your main files in your Public HTML Directory on your server the folder named PHP
  • Step 2 - Open the folder and open the file contact.php file with your prefered code editor.
  • Step 3 - Go to line 7 and after the words $sendTo = 'Description Here <yourmail@yourmail.com>';
  • Step 4 - Beware to put your mail between tags <hello@yourdomain.com> like so.
  • Step 5 - Make sure you save the file over FTP
  • Step 6 - Test your contact form to see if the mail goes through. If you struggle to make it work send a ticket to your server support, if everything else fails, send a ticket to uxithemes ticksy

Creating Pages #back to top

Creating pages in HTML is quite easy. For start you can click right on any HTML File in the FTP and you will see a option to Duplicate. If this doesn´t appear good 'ol copy and paste will do the trick

Establish your main home page. #top

  • Step 1 - Locate the main files in the public directory of your new awesome Theme.
  • Step 2 - Choose the html file you want to set as your home page. Eg: index-2.html, index-cookies.html, index.html etc.
  • Step 3 - If you want to leave index.html, don't change the name file
  • Step 4 - If you want another homepage, change the name of the file to index.html

Changing styles of elements#top

You need to open style.css first, on top, you can see main styles and main colors, If you want to change main color theme, open your prefered code editor, press CTRL+F or F3, then paste the hex main color (purple) #9B51E0; on the search bar. then you can see a button call replace all. On this field you must choose your color and then replace all.

How to target style specific elements

  • Step 1 - Open on your prefered Web Browser the page you want to style
  • Step 2 - Right click on any element you want to style, click Inspect Element a tab will open
  • Step 3 - Click a tab named "styles" you can see the set of styles for that particular element
  • Step 4 - You can change the style and live preview that style, as well you can see in the tab on what line is that style set
  • Step 5 - Once you got what you want, make the changes on the style.css and save the file over FTP

Attention: Many styles are defaulted by Bootstrap handy set of styles, if you want to override them, Make this override on style.css

Do not make changes on boostrap css files, you have been warned

How to Set Up the Instagram Feed #back to top

The Instagram Feed is one of the most interest characteristics of this Theme. I strongly suggest to use it, So here is a Guide to make your website shine with this feature.

For start, you have two main galleries, one that is the default one, it's in all footers, another one, is on blog pages as well as blog listing. So make sure you make changes on both to activate them.

Getting Instagram User ID and Access Token

  • Step 1 - Log in into your Instagram account on a web browser
  • Step 2 - Then go to https://www.instagram.com/developer/
  • Step 3 - At the top click on the Button that says "Manage Clients".
  • Step 4 - If you have not Register a new client.
  • Step 5 - Fill out everything and where it says redirect url put this url: http://instafeedjs.com
  • Step 6 - Then on the security tab make sure you uncheck "Disable implicit OAuth"
  • Step 7 - Finally use this link to authoruize Instafeed. Where it says "[Client ID]" replace that including the brackets with your clientID from instagram:
  • Step 8 - https://instagram.com/oauth/authorize/?client_id=[Client ID]&redirect_uri=http://instafeedjs.com&response_type=token
  • Step 9 - There you have the access Token, make sure you copy Access Token and the UserID, save it somewhere you remember and follow the next steps

Where to paste Client ID and Access Token

You are close! Now it is just copying and pasting on this following location

  • Step 1 - Open the folder js inside main directory.
  • Step 2 - Open the file main.js and then press Ctrl+f
  • Step 3 - Search for the word InstaFeed There you can see a function where the two feeds are activated. Where it says User Id paste your User Id between commas
  • Step 4 - Paste the Access Token between the two commas
  • Step 5 - There is a second function below the first one, paste them there too. You are done! Check it all works
